För att kunna arbeta med kursens kod och uppgifter på GitHub behöver du koppla ditt GitHub-konto till kursens organisation. 

På kursens [Canvas](https://canvas.bth.se) sida finns en uppgift som heter **GitHub invite**. Gå till den, följ instruktionerna där och lämna in uppgiften.

När uppgiften rättats och får du en inbjudan till organisationen. Du kommer att få en notis i Canvas och ett mejl till din GitHub användare (den du angav i uppgiften).

När du har fått inbjudan kan du acceptera den genom att klicka på länken i mejlet eller i Canvas. Vi rekommenderar att du använder kommentaren i Canvas. Där hittar du länk till både inbjudan och ditt repo.

## Aktivera Single Sign-On (SSO)

:::note
Om knappen `Single Sign-On` inte syns kan hoppa över detta steg.
:::

När du har accepterat inbjudan till organisationen behöver du aktivera Single Sign-On (SSO) för att kunna arbeta med kursens kod och uppgifter. Klicka på knappen `Single Sign-On` uppe till höger på sidan i organisationen.

## Aktivera SSH för organisationen

För att kunna arbeta med kursens kod och uppgifter på GitHub behöver du aktivera SSH för organisationen.

1. Uppe i högra hörnet på vilken sida som helst på GitHub, klicka på din profilbild och välj `Settings`.
2. I menyn till vänster, klicka på `SSH and GPG keys`.
3. För din SSH nyckel, klicka på "Configure SSO" och välj `bth-algo` i listan över organisationer.
