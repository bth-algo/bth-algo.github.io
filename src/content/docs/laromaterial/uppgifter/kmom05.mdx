### min-Heap

Du ska i denna uppgift implementera en min-Heap i klassen `Heap`. Klassen `Heap` ska implementeras baserat på det generiska interfacet `IHeap`.  
Både klassen `Heap` och interfacet `IHeap` ska finnas i namespace `Heap` och de ska vara generiska och de ska ha en typparameter T som har en begränsning så att den måste implementera interfacet `IComparable<T>` (samma som använts i samband med klassen BusStop). Interfacet `IHeap` hittar du nedan och detta skapar du själv i filen `IHeap.cs`.

```csharp
public interface IHeap<T> where T:IComparable<T>
{
    public void Insert(T item);
    public T Remove();
    public T Peek();
    public int Size();
    public T[] GetAllElements();
}
```

Metoden `GetAllElements` är egentligen inte en metod som ska ingå i en Heap men denna behövs för att vi ska kunna göra vissa kontroller när vi rättar din lösning.
Det är tillåtet och dessutom lämpligt att tillföra hjälpmetoder för att lösa vissa delproblem.

#### Sortera genom att använda en heap

Nu ska du använda din implementation av min-Heap för att utföra sortering. 

Stommen till metoden är

```csharp
public static void SortUsingAHeap<T>(T[] A, int n) where T : IComparable<T> {/*ska implementeras*/}
```

och denna ska placeras i en klass namngiven `UsingHeap` och även denna klass ska finnas i namespace `Heap`.

Sorteringsordningen ska vara från lägst till högst.

#### Kontrollera att din algoritm är korrekt

I Program.cs i ditt dotnet-projekt ska du skriva kod som kontrollerar att din sortering fungerar korrekt.

- Prova att sortera med de olika datatyperna `int`, `string` och `BusStop`. Tips: Använd t.ex filen `names.txt` som du laddade ner i kmom04 och skapa en array av strängar och filen `stops.txt` från kmom02 och skapa en array av `BusStop`-objekt som du sedan sorterar. 
- Överst i `Program.cs`skriver du en kommentar där du resonerar om tidskomplexiteten i ordo-notation för värsta fallet för din sortering.
- Kontrollera att arrayen verkligen är sorterad när algoritmen är klar. Du kan använda samma kod som användes i kmom02.

    <details>
    <summary>Den hittar du här</summary>

    ```csharp
    bool CheckSorted<T>(T[] arr, int n) where T: IComparable<T>
    {
        for (int i=0; i<n-1; i++)
        {
            if (arr[i].CompareTo(arr[i+1])>0)
                return false;
        }
        return true;
    }
    ```

#### Filer som ingår i inlämningen

I ditt dotnet-projekt ska följande filer ingå
- `IHeap.cs` - som innehåller interfacet IHeap
- `Heap.cs` - som innehåller klassen Heap som implementerar IHeap
- `UsingHeap.cs` - som innehåller klassen UsingHeap med metoden `SortUsingAHeap`
- `Program.cs`- där du har kod som kontrollerar att sorteringen fungerar korrekt och dessutom överst i filen, en kommentar med ditt resonemang över tidskomplexiteten i värsta fallet för din implementation av sorteringsalgoritmen.
