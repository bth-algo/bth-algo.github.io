### min-Heap

Du ska i denna uppgift implementera en min-Heap i klassen `Heap`. Klassen `Heap` ska implementeras baserat på det generiska interfacet `IHeap`

```csharp
public interface IHeap<T> where T:IComparable<T>
{
    public void Insert(T item);
    public T Remove();
    public T Peek();
    public int Size();
    public T[] GetAllElements();
}
```

Metoden `GetAllElements` är egentligen inte en metod som ska ingå i en Heap men denna behövs för att vi ska göra vissa kontroller när vi rättar din lösning.
Det är tillåtet och dessutom lämpligt att tillföra hjälpmetoder för att lösa vissa delproblem.

#### Sortera genom att använda en heap

Nu ska du använda din implementation av min-Heap för att utföra sortering. 

Stommen till metoden är

```csharp
public static void SortUsingAHeap<T>(T[] A, int n) where T : IComparable<T> {/*ska implementeras*/}
```

och denna ska placeras i en klass namngiven UsingHeap.

Sorteringsordningen ska vara från lägst till högst.

#### Kontrollera att din algoritm är korrekt

- Prova att sortera med olika datatyper (t.ex int och string).  

- Kontrollera att arrayen verkligen är sorterad när algoritmen är klar. Du kan använda samma kod som användes i kmom02.

### Hur effektiv är algoritmen

Nu ska du undersöka hur effektiva algoritmerna är genom att undersöka antalet jämförelser. Du ska göra detta genom att sortera BusStop-objekt.

Filen `stops.txt` innehåller hållplatser för Blekingetrafiken. Innehållet på filen är
```text
stop_id,stop_name,stop_lat,stop_lon,location_type,parent_station,platform_code
9021010001001000,Kungsplan,56.165202,15.586844,1,,
9021010001013000,Landsvägsgatan,56.172004,15.588238,1,,
9021010001014000,Tullskolan,56.174439,15.58903,1,,
9021010001015000,Polhemsgatan,56.176723,15.590546,1,,
...
```

Du kan använda följande kod för att läsa från fil och skapa en array med osorterade BusStop-objekt, där de 8 sista siffrorna i stop_id blir nummer (`_nr`) och stop_name blir namnet (`_name`).

Parametern `nrOf` motsvarar det antal som ska läsas från filen men om dess värde är -1 läses hela filen.

```csharp
BusStop[] ReadFromFile(string filename, int nrOf = -1)
{
    try 
    {
        string[] stops = File.ReadAllLines(filename);
            if (nrOf == -1 || nrOf > stops.Length)
        nrOf = stops.Length - 1;
        BusStop[] busStops = new BusStop[nrOf];
        for (int i=0; i<nrOf; i++)
        {
            string[] parts = stops[i+1].Split(",");
            busStops[i] = new BusStop(parts[1], int.Parse(parts[0].Substring(8)));
        }
        return busStops;
    }
    catch(Exception e)
    {
        return null;
    }
}

```

Obs: För att resultatet av sortering av de osorterade BusStop-objekten ska blir relevant behöver du göra minst 5 olika exekveringar med olika ordning i arrayerna. Ett tips är att blanda objekten inför varje ny sortering. När dessa körningar genomförts bestämmer du medelvärdet av antalet jämförelser.

{
    for (int times=1; times<n; times++)
    {
        int i = random.Next(0, n);
        int j = random.Next(0, n);
        (arr[i], arr[j]) = (arr[j], arr[i]);
    }
}
