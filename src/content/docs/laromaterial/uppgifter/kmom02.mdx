### Sorteringsalgoritmer

Algoritmerna ska vara generiska men kräva att elementen som sorteras implementerar   
`IComparable`-gränssnittet så att de kan jämföras med varandra.

Du ska implementera insertionsort och quicksort. Dessa ska finnas i filen SortingAlgorithms.cs som ska tillhöra namespace `Sorting`. 

#### Insertionsort

Stommen till Insertionsort ska vara följande

```csharp
public static void Insertionsort<T>(T[] A, int n) where T : IComparable<T> {/*ska implementeras*/}

```

#### Quicksort

Quicksort ska finnas i två överlagrade metoder där en är publik och en är privat enligt följande  

```csharp
public static void Quicksort<T>(T[] A, int n) where T : IComparable<T> {/*ska implementeras*/}
private static void Quicksort<T>(T[] A, int start, int end) where T : IComparable<T> {/*ska implementeras*/}

```
Den privata metoden är den som är rekursiv och denna anropas från den publika metoden. 

Partitioneringsalgoritmen där sista elementet är pivot får du här:

```csharp 
private static int Partition<T>(T[] A, int start, int end) where T : IComparable<T>
{
    T pivot = A[end];
    int i = start - 1;
    
    for (int j = start; j < end; j++)
    {
        if (A[j].CompareTo(pivot) < 0)
        {
            i++;
            (A[i], A[j]) = (A[j], A[i]);
        }
    }
    (A[i + 1], A[end]) = (A[end], A[i + 1]);
    return i + 1;
}
```

#### Kontrollera att dina algoritmer är korrekta

- Prova att sortera med olika datatyper (t.ex int och string).  

- Kontrollera att arrayen verkligen är sorterad när algoritmen är klar.

### Använd sorteringsalgoritmerna

Nu ska du undersöka hur effektiva algoritmerna är genom att undersöka antalet jämförelser. Du ska göra detta genom att sortera BusStop-objekt.

Filen `stops.txt` (som du hittar i katalogen `example`) innehåller hållplatser för Blekingetrafiken. Innehållet på filen är
```text
stop_id,stop_name,stop_lat,stop_lon,location_type,parent_station,platform_code
9021010001001000,Kungsplan,56.165202,15.586844,1,,
9021010001013000,Landsvägsgatan,56.172004,15.588238,1,,
9021010001014000,Tullskolan,56.174439,15.58903,1,,
9021010001015000,Polhemsgatan,56.176723,15.590546,1,,
...
```

Du kan använda följande kod för att läsa från fil och skapa en array med osorterade BusStop-objekt, där de 8 sista siffrorna i stop_id blir nummer (`_nr`) och stop_name blir namnet (`_name`).

Parametern `nrOf` motsvarar det antal som ska läsas från filen men om dess värde är -1 läses hela filen.

```csharp
BusStop[] ReadFromFile(string filename, int nrOf = -1)
{
    try 
    {
        string[] stops = File.ReadAllLines(filename);
            if (nrOf == -1 || nrOf > stops.Length)
        nrOf = stops.Length - 1;
        BusStop[] busStops = new BusStop[nrOf];
        for (int i=0; i<nrOf; i++)
        {
            string[] parts = stops[i+1].Split(",");
            busStops[i] = new BusStop(int.Parse(parts[0].Substring(8)), parts[1]);
        }
        return busStops;
    }
    catch(Exception)
    {
        return Array.Empty<BusStop>();
    }
}

```

```csharp
void Shuffle<T>(T[] arr, int n) where T: IComparable<T>
{
    Random random = new Random();
    for (int times=1; times<n; times++)
    {
        int i = random.Next(0, n);
        int j = random.Next(0, n);
        (arr[i], arr[j]) = (arr[j], arr[i]);
    }
}

```
